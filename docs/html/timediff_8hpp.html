Detailed Description

Computes several time differences between different time scales.

Calculates the differences between Universal Time, International Atomic Time, Coordinated Universal Time, GPS Time, and Terrestrial Time, returning them as a tuple.

Parameters
UT1_UTCDifference between UT1 and UTC in seconds.
TAI_UTCDifference between TAI and UTC in seconds.
Returns
A tuple containing:
  • UT1_TAI: difference between UT1 and TAI [s]
  • UTC_GPS: difference between UTC and GPS [s]
  • UT1_GPS: difference between UT1 and GPS [s]
  • TT_UTC: difference between TT and UTC [s]
  • GPS_UTC: difference between GPS and UTC [s]
