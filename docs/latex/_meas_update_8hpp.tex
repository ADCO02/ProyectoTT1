\doxysubsection{Detailed Description}
Performs a Kalman filter measurement update step. 

Implements the measurement update step of a Kalman filter. Given the prior state and covariance, along with a new measurement and its expected value and sensitivity, it computes the Kalman gain, updates the state vector and covariance matrix accordingly.
