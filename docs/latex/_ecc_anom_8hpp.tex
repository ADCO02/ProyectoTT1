\doxysubsection{Detailed Description}
Computes the eccentric anomaly for elliptic orbits. 

Solves Kepler\textquotesingle{}s equation using iterative Newton-\/\+Raphson method to find the eccentric anomaly corresponding to a given mean anomaly.
