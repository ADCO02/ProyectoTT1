\doxysubsection{Detailed Description}
Computes Greenwich Apparent Sidereal Time (GAST) from UT1. 

Returns the Greenwich Apparent Sidereal Time (GAST) in radians, based on the input Modified Julian Date in the UT1 time scale. GAST accounts for both the Earth\textquotesingle{}s rotation (GMST) and the equation of the equinoxes, reflecting the apparent position of the vernal equinox due to nutation.
