\doxysubsection{Detailed Description}
Transformation from true equator and equinox to Earth equator and Greenwich meridian system. 

Generates the Greenwich Hour Angle (GHA) rotation matrix based on the Greenwich Apparent Sidereal Time at the given UT1 date.
