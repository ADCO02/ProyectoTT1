\doxysubsection{Detailed Description}
Computes associated Legendre polynomials and their derivatives. 

Given degree n, order m, and angle fi (in radians), this function computes the normalized associated Legendre functions ( P\+\_\+\{nm\}(\textbackslash{}phi) ) and their derivatives with respect to (\textbackslash{}phi).
