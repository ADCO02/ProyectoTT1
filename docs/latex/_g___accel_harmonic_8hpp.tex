\doxysubsection{Detailed Description}
Computes the gradient of the Earth\textquotesingle{}s harmonic gravity field. 

Calculates the gradient matrix of the acceleration vector generated by the Earth\textquotesingle{}s harmonic gravity field at a given satellite position, using a gravity model up to specified degree and order.
